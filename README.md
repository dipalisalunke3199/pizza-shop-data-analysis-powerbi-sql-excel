# Pizza Shop Sales Analysis Power BI Project

![Dashboard Screenshot](Visualization in Power BI.png)
# Project Overview  
This project analyzes 1 year of pizza shop sales data to understand the reason behind declining revenue and identify opportunities for business improvement using data-driven insights and Power BI visualization. The main objective is to improve sales performance, customer satisfaction, and operational efficiency.

# Problem Statement  
The pizza shop has experienced a decline in revenue. The business needed a detailed analysis of historical order data to identify reasons for sales decline, customer behavior patterns, best and worst performing pizzas, and opportunities for improvement in pricing, promotion, and operations.

# Project Objectives  
- Analyze sales performance over time  
- Identify best-selling and worst-selling pizzas  
- Understand customer buying patterns  
- Improve revenue and profitability  
- Optimize operations and staffing  
- Support decision-making using dashboards  

# Tools and Technologies Used  
- Power BI for data visualization and dashboard creation  
- Microsoft Excel for data cleaning and preparation  
- Power Query for data transformation  
- DAX for basic calculations and measures  

# Methodology  

# Data Collection  
Historical sales data from pizza shop for 1 year provided in Excel format with multiple sheets.

# Data Cleaning  
- Removed null and blank values  
- Standardized date and time formats  
- Merged multiple datasets into a single table  

# Data Transformation  
- Created calculated columns such as Month and Hour  
- Applied Power Query transformations  
- Built relationships between tables in Power BI  

# Data Analysis  
- Identified sales trends  
- Analyzed peak order hours  
- Determined best-selling and worst-selling pizzas  
- Calculated total revenue and average order value  

# Visualization  
- Built interactive Power BI dashboard  
- Created KPI cards, charts, and trend graphs  
- Highlighted key business insights  

# Project Scope (5P Framework)  

# Product  
- Identify best and worst-selling pizzas  
- Improve low-performing products using popular ingredients  
- Introduce combo offers  

# Price  
- Implement dynamic pricing strategies  
- Introduce combo meals to increase order value  

# Promotion  
- Discounts on low-selling pizzas  
- Seasonal and festive offers  
- Off-peak promotions and events  

# Place  
- Understand customer demand patterns by location  
- Identify expansion opportunities  

# People  
- Optimize staff scheduling during peak hours  
- Improve operational efficiency  

# Key Insights  
- Peak ordering hours identified for better staffing  
- Few pizza types contribute to majority of revenue  
- Some products consistently underperform  
- Customer demand varies by time and season  
- Combo orders increase average order value  

# Recommendations  
- Improve worst-selling pizzas using popular ingredients  
- Introduce combo meal offers to increase revenue  
- Focus marketing on high-demand time slots  
- Optimize staff allocation during peak hours  
- Use discounts strategically during low-sales periods  
- Remove or redesign underperforming menu items  

# Challenges Faced  
- Data inconsistency across multiple sheets  
- Missing and null values in raw dataset  
- Time-based analysis required data transformation  
- Building relationships in Power BI  
- Designing a clean and interactive dashboard  

# Dashboard  
A Power BI dashboard was created to visualize total revenue, total pizzas sold, best and worst sellers, peak order timings, and sales trends over time.
